# Tanvi Jain — Portfolio Site

A single-page, animated portfolio built from your resume.

## How to publish on GitHub Pages

1. Create a new GitHub repository (e.g. `tanvi-jain` or `yourusername.github.io`).
2. Add `index.html` to the root of the repo.
3. Add your photo to the **same folder** as `index.html`, named exactly **`profile.jpg`**
   (or edit the `src="profile.jpg"` line near the top of `index.html` to match your filename).
4. In the repo settings → **Pages**, set the source branch to `main` (root folder) and save.
5. Your site will be live at `https://yourusername.github.io/repo-name/` within a minute or two.

## Before you publish — 2 things to update

Open `index.html`, go to the **Contact** section near the bottom, and replace these placeholders:

- `https://linkedin.com/in/your-handle-here` → your real LinkedIn URL
- `https://instagram.com/your-handle-here` → your real Instagram URL

(WhatsApp, email, and phone are already wired to your real details from the resume.)

## What's inside

- Ledger/passbook-styled visual theme (ruled lines + red margin, nodding to your CA world)
- Scroll-reveal animations, animated stat counters, sticky nav with active-section highlighting
- Fully responsive, with a mobile menu
- All content — education, both roles, skills, personal details — pulled directly from your resume
